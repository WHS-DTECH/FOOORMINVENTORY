Route Descriptions

  • recipes: View and manage all recipes (full recipe management page)
  • recbk: Recipe Book - Browse recipes in grid or table view
  • class_ingredients: Class ingredients calculator and ingredient management
  • booking: Class booking system for scheduling
  • shoplist: Shopping list generator for classes
  • admin: Administrative functions (staff/class upload, recipe management, permissions)

⚠️ Important Notes

  • VP role should always have access to all pages including admin
  • Admin page access should be restricted to trusted roles only
  • Changes take effect immediately
  • Users will see "Access Denied" if they try to access a page they don't have permission for
  • The public role applies to non-authenticated users
